    as tags utlizadas  <picture> <source> 
    ‣ type vai indicar o media type da imagem que usamos. 
    ‣ srcset vai configurar o nome da imagem que será carregada quando o tamanho indicado for atingido.
    ‣ media indica o tamanho máximo a ser considerado para carregar a imagem
indicada no atributo srcset.

Você pode até colocar o valor exato de 1000px na
propriedade max-width, mas vai perceber que um valor ligeiramente
acima vai gerar resultados mais interessantes.
